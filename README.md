# Netflix Clone

Netflix clone with Netflix like UI and functionalities built using React native and AWS

![Logo](https://upload.wikimedia.org/wikipedia/commons/0/08/Netflix_2015_logo.svg)


  

Install dependencies

```bash
  npm install
  yarn
  expo install
```

Start the server

```bash
  expo start
```
